Technologies used

Readme first ››

Masas is built using different technologies. Some of them are:

  • ESRI Javascript API version 3.15 compact
  • JQuery 1.11.3
  • JQuery Mobile 1.4.4
  • JQuery UI 1.10
  • Modernizr.js library for detecting tags in old browsers
  • Proj4.js library for transforming between different SRS
  • Microsoft SQL Server 2014
  • Personalize Web Services


A proper combination of all of these technologies to ease the solution here shown.
